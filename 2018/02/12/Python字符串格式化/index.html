Python 字符串格式化

发表于 2018-02-12 | 更新于: 2018-02-12 | 分类于 Python | | 阅读次数:

Python 有两种格式化字符串的方式，我们习惯简单地称之为旧式(old style)和新式(new style)。这两种方式在 Python 2 和 Python 3 中都适用(新式格式化方法适用于 Python 2.6 及 以上)。

使用%的旧式格式化

旧式格式化的形式为 string % data。其中 string 包含的是待插值的序列。

完整语法格式：

  • (name)：可选，用于选择指定的key

  • flags：可选，可供选择的值有:

    | flag | 含义 |
    | —- | ————————————————- |
    | + | 右对齐；正数前加正好，负数前加负号； |
    | - | 左对齐；正数前无符号，负数前加负号； |
    | 空格 | 右对齐；正数前加空格，负数前加负号； |
    | 0 | 右对齐；正数前无符号，负数前加负号；用0填充空白处 |

  • width： 可选，占有宽度

  • .precision：可选，小数点后保留的位数

  • typecode：必选（参见下表）
